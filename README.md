# Handlebars lipsum
Handlebars helper that returns a substring of characters of lorem ipsum latin text.

## Usage
```javascript
{{lipsum 0 300}}
```

## Parameters

| Argument	| Type 		| Description								|
|:----------|:----------|:------------------------------------------|
| from		| Number 	| The starting index.						|
| to 		| Number 	| The ending index.							|
